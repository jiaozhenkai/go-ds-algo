# go-ds-algo

go 实现常用的数据结构与算法

```
go get -u github.com/jiaozhenkai/go-ds-algo
```

使用 stack:

```
import github.com/jiaozhenkai/go-ds-algo/linear-structure/stack/arraystack
```

使用 queue

```
import github.com/jiaozhenkai/go-ds-algo/linear-structure/queue/arrayqueue
```
